##### Visuel
- Router Grund Infos

	- Geräte hat das Wlan verlassen

#### Geräte 

##### Visuel
Visuel:

- Liste aller verbundenen Geräte
	- Mit Icons versehen
	
- Geräte Verlauf
	  - Gerät XY war hat sich zu letzt vor x Stunden verbunden
	    
- Aktionen
	- Gerät pausieren(Internet Blockieren)
	- Gerät Priorisieren (QoS)
	- Name bearbeiten
	- Blockliste für dieses Gerät einschalten/ausschlaten
- Details pro Gerät
	- IP/Mac Adresse
	- Verbindungstyp (LAN/WLAN)
	- Bandbreite des Gerätes
	- Profil des Geräts (Gast, Kind, unklassifiziert)
