Projektinformationen

Moderne, einfache und übersichtliche Netzwerkverwaltung mit zusätzlichen Features für Normalverbraucher

  • Projektstart: 2025
  • Technologien: OpenWRT, IPTables, BusyBox (Router); HTML, CSS, JS/TS (Frontend); Quarkus, PostgreSQL DB (Cloud)
  • Zielgruppe: Familien, Technikinterressierte

Im folgenden Teil wird das Systemdesign und die Funktionalität vorgestellt.
